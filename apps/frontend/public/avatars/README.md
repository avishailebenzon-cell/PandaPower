# Agent Avatars

Drop one real (AI-generated) photo per agent here. The frontend loads each photo
from `/avatars/<code>.jpg`. Until a file exists, the UI automatically falls back
to the original "initial in a colored circle" SVG (no broken images).

## Requirements
- Format: **.jpg** (filename must match the code exactly, lowercase)
- Square, ideally **512×512** or larger, face centered (rendered as a circle)
- Realistic headshot, neutral/office background
