                         PANDAPOWER CLOUD DEPLOYMENT
                             Vercel + Render Setup
═══════════════════════════════════════════════════════════════════════════════

📋 CHECKLIST - DO THIS IN ORDER:

☐ STEP 1: GitHub Setup (5 minutes)
   1. Go to https://github.com/new
   2. Create repository: "pandapower"
   3. Copy the commands from GitHub
   4. Run locally:
      
      git branch -M main
      git remote add origin https://github.com/YOUR-USERNAME/pandapower.git
      git push -u origin main
   
   ✅ Verify: Check your GitHub repo has the code

☐ STEP 2: Deploy Frontend to Vercel (10 minutes)
   1. Go to https://vercel.com
   2. Sign in with GitHub
   3. Import your "pandapower" repository
   4. Choose: Framework = "Other"
   5. Root directory = "apps/frontend"
   6. Build command = "npm run build"
   7. Output directory = "dist"
   8. Click Deploy
   
   ✅ You'll get a URL: https://pandapower-frontend-xxx.vercel.app

☐ STEP 3: Set Frontend Environment Variables (2 minutes)
   1. Go to Vercel dashboard → pandapower-frontend project
   2. Settings → Environment Variables
   3. Add new variable:
      Key: VITE_API_URL
      Value: https://pandapower-backend.onrender.com
   4. Redeploy

☐ STEP 4: Deploy Backend to Render (25 minutes)
   1. Go to https://render.com
   2. Sign in with GitHub
   3. Click "New" → "Web Service"
   4. Select: pandapower repository
   5. Configure:
      - Name: pandapower-backend
      - Root Directory: apps/backend
      - Build Command: pip install -r requirements.txt
      - Start Command: python -m uvicorn pandapower.main:app --host 0.0.0.0 --port $PORT
      - Plan: Free (for now)
   6. Add Environment Variables:
      PYTHONPATH=/opt/render/project/apps/backend/src
      SUPABASE_URL=<from your .env>
      SUPABASE_KEY=<from your .env>
      SUPABASE_JWT_SECRET=<from your .env>
      ANTHROPIC_API_KEY=<from your .env>
      RESEND_API_KEY=<from your .env>
   7. Click "Create Web Service"
   
   ⏳ Wait 10-15 minutes for deployment
   
   ✅ You'll get a URL: https://pandapower-backend.onrender.com

☐ STEP 5: Deploy Celery Worker (5 minutes)
   1. In Render: New → Background Worker
   2. Configure:
      - Name: pandapower-worker
      - Root Directory: apps/backend
      - Build Command: pip install -r requirements.txt
      - Start Command: celery -A pandapower.workers.celery_app worker -l info
   3. Add same environment variables as Web Service
   4. Click "Create Background Worker"
   
   ⏳ Wait 5 minutes

☐ STEP 6: Deploy Celery Beat Scheduler (5 minutes)
   1. In Render: New → Cron Job
   2. Configure:
      - Name: pandapower-beat
      - Schedule: 0 * * * * (every hour)
      - Root Directory: apps/backend
      - Build Command: pip install -r requirements.txt
      - Start Command: celery -A pandapower.workers.celery_app beat -l info
   3. Add same environment variables
   4. Click "Create Cron Job"
   
   ⏳ Wait 5 minutes
